Privacy Policy

Effective Date: March 9, 2026

CryptoTalk values your privacy and is committed to protecting your personal information. This Privacy Policy explains how the CryptoTalk mobile application collects, uses, and safeguards your information when you use the app.

1. Information We Collect

Account Information

When you create an account in CryptoTalk, we collect your email address and authentication credentials through Firebase Authentication. This information is used only for creating and managing your account.

Message Data

Messages sent through CryptoTalk are protected with End-to-End Encryption (E2EE). Messages are encrypted on your device before being sent and can only be decrypted by the intended recipient. The server stores only encrypted message data and cannot read or access the message contents.

Technical Information

We may collect limited technical information such as device type, operating system version, and application version. This information helps improve the performance and stability of the application.

2. How We Use Your Information

CryptoTalk uses collected information to:

  • Create and manage user accounts
  • Enable secure messaging between users
  • Improve application performance and reliability
  • Maintain system security and prevent misuse

CryptoTalk does not sell, rent, or trade personal information to third parties.

3. End-to-End Encryption

CryptoTalk uses strong end-to-end encryption to protect all user messages. Encryption ensures that messages are encrypted before leaving the sender's device and can only be decrypted by the receiver's device.

Encryption keys are stored securely on the user’s device using Android security mechanisms such as the Android Keystore system.

4. Data Storage

Encrypted messages and related metadata are stored using secure cloud services such as Firebase Firestore. Since all messages are encrypted before being uploaded, the server cannot access the actual message content.

5. Data Security

CryptoTalk implements modern security practices to protect user data, including:

  • End-to-End Encryption
  • Secure storage of private keys using Android Keystore
  • Encrypted cloud database storage
  • Secure authentication using Firebase Authentication

These measures help prevent unauthorized access to user data.

6. Third-Party Services

CryptoTalk uses trusted third-party services to operate the application:

  • Firebase Authentication – for secure account login
  • Firebase Firestore – for encrypted message storage and real-time database functionality

These services may collect limited information in accordance with their own privacy policies.

7. Children’s Privacy

CryptoTalk is not intended for children under the age of 13. The application does not knowingly collect personal information from children. If such information is discovered, it will be removed promptly.

8. Changes to This Privacy Policy

CryptoTalk may update this Privacy Policy from time to time. Any changes will be posted within the application or on the official website. Users are encouraged to review the policy periodically.
